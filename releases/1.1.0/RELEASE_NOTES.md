        # Release Notes — 1.1.0

        ## Özet
        Flutter tabanlı iOS eğitim uygulamasını premium 2026 ürün kalitesine taşıma brief’i kuruldu. Screen-by-screen redesign, iOS-native his, component sistemi ve calm/premium visual language tanımlandı. Geliştiriciye uygulanabilir frontend modernizasyon sırası verildi.

        ## Yeni sayılan ana geliştirmeler
        - Brutally detail-oriented design director tavrı.
- Question-solving app odaklı UX yeniden inşası.
- Component, spacing, typography ve state disiplininin sertleştirilmesi.
