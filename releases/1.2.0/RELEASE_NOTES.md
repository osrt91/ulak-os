        # Release Notes — 1.2.0

        ## Özet
        Prompt artık bir işletim sistemi olarak ele alınmaya başlandı. Kapsam matrisi, overlay mantığı, security/QA/release/compliance katmanları ve governance bölümleri büyüdü. Feature flag, supply chain, app-store dağıtımı, schema-first outputs ve MCP düşüncesi çekirdeğe girdi.

        ## Yeni sayılan ana geliştirmeler
        - Mode selection engine ve evrensel kapsam matrisi.
- Frontend + mobile + premium redesign engine entegrasyonu.
- Distribution, store, compliance, release readiness ve supply-chain katmanları.

        ## Dağıtım notu
        Bu sürüm, Ulak OS eşit dağıtım politikasında standart release klasör yapısıyla taşınır.
