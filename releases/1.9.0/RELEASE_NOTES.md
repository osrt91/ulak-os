        # Release Notes — 1.9.0

        ## Özet
        Ulak OS adı altında GitHub’a koyulabilir repo yapısı üretildi. Claude Code, Codex/Copilot ve Gemini için adaptör dosyaları aynı çekirdeğe bağlandı. History pack, GitHub repo zip ve bootstrap ile yayınlanabilir dağıtım adımı atıldı.

        ## Yeni sayılan ana geliştirmeler
        - ulak-os_1.9.0_history_pack.zip
- ulak-os_1.9.0_github_repo.zip
- ulak_os_1_9_0_bootstrap.py
- vendor adapter docs

        ## Dağıtım notu
        Bu sürüm, Ulak OS eşit dağıtım politikasında standart release klasör yapısıyla taşınır.
