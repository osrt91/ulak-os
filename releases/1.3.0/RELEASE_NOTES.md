        # Release Notes — 1.3.0

        ## Özet
        Claude Code execution-first pack mantığı tanımlandı. CLAUDE.md, .claude/skills, agents, commands, hooks, MCP ve evals topolojisi netleştirildi. Skill authoring, plugin discipline ve subagent isolation ilk kez first-class hale geldi.

        ## Yeni sayılan ana geliştirmeler
        - Repo topolojisi ve pack generation fikri.
- Third-party skill/plugin adoption rubric.
- Execution protocol, change batching ve controlled editing mantığı.
