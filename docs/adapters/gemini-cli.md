# Gemini CLI adapter

## Başlangıç
1. `/memory reload`
2. `/commands reload`
3. Gerekirse `/memory show`

## Kullanım modeli
- `GEMINI.md` proje bağlamını verir.
- `.gemini/commands/*.toml` proje komutlarını sağlar.
- İstek tam program ise `/director komple` ile başlat.

## Uygun komutlar
- `/director komple`
- `/intake brownfield`
- `/market-scan rakipler`
- `/frontend:war-room onboarding`
- `/final-verdict`

## Beklenen çıktı
Gemini de aynı artefakt zincirini izlemeli ve validation olmadan done dememeli.
