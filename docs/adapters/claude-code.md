# Claude Code adapter

## Önerilen kullanım
- Claude Code repo kökünde açılır.
- `CLAUDE.md` başlangıç belleğidir.
- `.claude/commands/` içindeki komutlar çalıştırılır.

## Başlangıç komutları
- `/director komple`
- `/intake brownfield audit`
- `/frontend-war-room soru çözme ekranı`
- `/pack-gap-audit`
- `/final-verdict`

## Beklenen davranış
- gerekli subagent’ları seç
- `reports/current/` altında artefaktları sırayla yaz
- gerekiyorsa skill/command/agent gap’lerini açıkça raporla
